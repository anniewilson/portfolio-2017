# Annie Wilson - Portfolio site

## Requirements
 
- [Node.js](https://nodejs.org/en/)
- [Wintersmith](https://github.com/jnordberg/wintersmith)

## Setup

```
  $ npm install
```

## Development

```
  $ npm start
```

Website will be served from `http://localhost:8080/`

## Build

```
  $ npm run build
```
